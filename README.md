# BlogBurst - AI Marketing Agent for OpenClaw

Your autonomous AI marketing agent. Auto-posts to 6 platforms, engages with your audience, learns what works, and optimizes your strategy 24/7.

**Not a scheduler. Not a writing tool. A full AI marketing team.**

## What It Does

- **Auto-Pilot**: Posts 3x/day to Twitter, Bluesky, Telegram, Discord — you do nothing
- **Smart Engagement**: Auto-replies to mentions, likes relevant posts, follows target accounts
- **Self-Learning**: Tracks performance, runs A/B tests, gets smarter every day
- **SEO + GEO**: Optimizes for Google AND AI search engines (ChatGPT, Perplexity, Gemini, Claude)
- **Multi-Industry**: Works for SaaS, apps, agencies, creators, e-commerce — any product

50+ countries, 7 active auto-pilot users, 300+ posts published. Free tier: 50 posts/month.

## Installation

```bash
npx clawhub@latest install blogburst
```

Or manually:
```bash
cd ~/.openclaw/skills
git clone https://github.com/shensi8312/blogburst-openclaw-skill.git blogburst
```

## Setup (2 minutes)

1. Sign up free at [blogburst.ai](https://blogburst.ai)
2. Add your product (tell the AI what you're marketing)
3. Connect Twitter/Bluesky (one-click OAuth)
4. Get API key: Dashboard > API Keys
5. Set environment variable:

```bash
export BLOGBURST_API_KEY="your-api-key"
```
